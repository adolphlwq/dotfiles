# dotfiles

A tool for configing vim,zsh,etc

## Usage
- clone repo
```
git clone https://github.com/adolph/dotfiles
```
- update submodules
```
git submodules update --init
OR
make submodules
```
- vim
```
make vim
```
- zsh
```
make zsh
```
- git
```
make git
```
- vscode
```
make vscode
```
